# CalculusTool
## О проекте
Проект предоставляет инструменты для символьных математических вычислений с использованием библиотеки SymPy. Позволяет вычислять производные функций, включая производные высших порядков. Разработан для студентов, преподавателей и всех, кто работает с математическим анализом.
## ✨ Функциональность 
- Вычисление производных первого порядка
- Вычисление производных n-го порядка (вторые, третьи и последующие производные)
- Поддержка символьных вычислений
- Обработка ошибок ввода
## 🛠 Требования/зависимости
- Python 3.14.0
- Зависимости: смотрите `requirements.txt`

# Установка

## Клонирование репозитория
git clone https://github.com/sofya-kuchinskaya/CalculusTool.git

cd CalculusTool

## Создание виртуального окружения
### Для Windows:
python -m venv calculus_venv

calculus_venv\Scripts\activate

### Для macOS/Linux:
python3 -m venv calculus_venv

source calculus_venv/bin/activate

### Установка зависимостей
pip install -r requirements.txt или pip3 install -r requirements.txt

## Запуск
python calculus_core.py
